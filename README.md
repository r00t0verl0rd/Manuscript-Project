# Manuscript Project

Демо‑веб‑страница с «неоновой» атмосферой и лентой идей (для людей и для роботов). Проект стилизован на CSS (включая матовое стекло, неоновые эффекты, адаптив под мобильные), а фон — анимация **particles.js**.

> ⚠️ В текущей версии работа с блокчейном/ончейн‑частью (mint/голосование) отключена: интерфейс работает как демо.

## Что внутри

- **index.html** — структура страницы: подключение кошелька (кнопка), форма отправки идеи, лента идей с двумя колонками.
- **script.js** — демо‑логика: перевод интерфейса (RU/EN), отрисовка идей, сортировка по лайкам, заглушки для действий.
- **styles.css** — основной стиль: нейронный/киберпанк дизайн, карточки со скошенными углами, кнопки и анимации.
- **particles-config.js** — настройка фона particles.js.

## Функциональность

### Языки интерфейса

В шапке доступны переключатели **RU / EN**. Тексты подменяются на лету через `data-i18n`.

### Лента идей

Идеи разделены на две колонки:

- **Legacy: Humans 🧬** — идеи «для людей»
- **Future: Robots 🤖** — идеи «для роботов»

Список в демо рендерится из массива `mockIdeas` и сортируется по `likes`.

### Подключение кошелька / mint

Кнопка «Connect» присутствует в интерфейсе. Однако в коде создание идеи и запись в блокчейн отключены — при отправке формы показывается уведомление.

## Установка и запуск

Так как проект — статический, запустить можно локально в браузере.

1. Откройте `index.html` в браузере.
2. Лента и части интерфейса загрузятся автоматически.

Также можно быстро запустить локальный сервер (если используете, например, VS Code Live Server).

## Используемые технологии

- HTML / CSS / Vanilla JavaScript
- particles.js (подключается из CDN)
- Google Fonts (Comfortaa, Russo One)
